# 🎲 Random GIF Generator

A simple and fun web application that generates random GIFs — including a dedicated section for **Salman Khan GIFs** 😄

---

## 🚀 Live Demo

🔗 https://random-gif-generator-h1mf.vercel.app/

---


## ✨ Features

- 🎯 Generate a completely random GIF  
- 🔍 Generate Salman Khan GIFs using tag search  
- ⚡ Fast and responsive UI  
- 🎨 Clean and minimal layout  
- 🔁 One-click GIF refresh  

---

## 🛠️ Tech Stack

- Frontend: React.js  
- Styling: CSS / Tailwind CSS  
- API: Giphy API  
- HTTP Client: Axios / Fetch  

---
## 🧠 How It Works

### 🔹 Random GIF
- Fetches a random GIF using Giphy API  
- Displays a new GIF on each button click  

### 🔹 Tag-based GIF
- Takes user input (e.g., "Salman Khan")  
- Fetches related GIFs dynamically  

### 🔹 Custom Hook (`useGif`)
- Handles API calls  
- Manages loading state  
- Reusable across components
